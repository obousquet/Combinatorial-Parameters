Majority

Name: Majority

Symbol:
\text{Maj}
Definition:

(see \cite{maass1992lower}): ${S\subset [n+1]: n+1\in S\Leftrightarrow |S\backslash {n+1\}|>n/2}$

So we have all subsets of $[n]$ and an extra majority bit.
