Log Size

Name: Log Size

Symbol: $\log \mathrm{H}$

Definition:

The Log Size of a family $\mathcal{H}$ is simply the (binary) log of its cardinality, $$ \log |\mathcal{H|} $$

Category: Basic

Symmetric: Yes

Monotonic: Yes

P-Monotonic: Yes

Doubly Monotonic: Yes

Strictly Monotonic: Yes

Values:

  • Log Size of Addressing
  • Log Size of Majority
  • Log Size of Tagged Singletons
  • Log Size of Half-intervals
  • Log Size of Singletons plus empty set
  • Log Size of Full Cube
  • Log Size of Singletons
  • Log Size of Trivial concepts
