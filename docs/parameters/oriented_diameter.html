Oriented Diameter

Name: Oriented Diameter

Symbol: $\mathrm{D}^+$

Definition:

The oriented diameter $\mathrm{D}^+(\mathcal{H})$ of a family $\mathcal{H}$ is defined as $$ \mathrm{D}^+(\mathcal{H}):= \max{|S|: {\emptyset, S}\subset \mathcal{H}_{|S}} $$

Category: Basic

Symmetric: No

Monotonic: Yes

P-Monotonic: Yes

Doubly Monotonic: Yes

Strictly Monotonic: No

Values:

  • Oriented Diameter of Singletons plus empty set
  • Oriented Diameter of Singletons
  • Oriented Diameter of Trivial concepts
  • Oriented Diameter of Tagged Singletons
  • Oriented Diameter of Full Cube
  • Oriented Diameter of Half-intervals

Relationships (as Parameter 1):

  • Oriented Diameter / Threshold Dimension
  • Oriented Diameter / Diameter
  • Oriented Diameter / Hamming Radius

Relationships (as Parameter 2):

  • Diameter / Oriented Diameter
  • Hamming Radius / Oriented Diameter
